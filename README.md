# DiscorServer

Servidor estilo Discord com:
- Chat em tempo real (simulado)
- Chamadas de voz (interface)
- Perfis animados
- Múltiplos idiomas (PT, EN, ES, FR)
- Tema preto, branco e vermelho

## Como hospedar no GitHub Pages
1. Crie um repositório público.
2. Envie todos os arquivos acima.
3. Ative o GitHub Pages nas configurações (branch main, pasta /).
4. Acesse `https://seu-usuario.github.io/DiscorServer/`

## Tecnologias
- HTML5, CSS3, JavaScript
- Font Awesome
- Sem dependências externas
